ReadySetStores - איפיון פרוייקט

פלטפורמה לבניית וניהול חנויות אונליין

1. סקירה טכנית

מערכת SPA ב-React 18 ו-Next.js 14 עם שילוב SSR/CSR לאופטימיזציית SEO וטעינה מהירה.
עיצוב Frontend ב-Tailwind CSS, ו-Backend ב-Firebase Auth, Firestore ו-Node.js 22.

2. מטרות מרכזיות

  • זמני טעינה <2 שניות (Lighthouse ≥90).
  • דפי מוצר דינמיים בעזרת SSG/ISR של Next.js.
  • Scale הוריזונטלי מלא ב-Vercel/Railway.

3. תכונות עיקריות

  • תבניות UI מותאמות אישית עם TailwindCSS ו-React (TypeScript)
  • ניהול מוצרים מושלם: הוספה/עריכה/מחיקה עם תמיכה בתמונות ראשיות ומשניות, וריאנטים, קטגוריות, מחיר בסיס, מלאי בזמן אמת, תיאורי מוצר מפורטים באמצעות עורך Rich Text (כותרות, הדגשה, רשימות, קישורי מדיה), יבוא/ייצוא CSV ואודיט לוג – כל הנתונים מאוחסנים ב-PostgreSQL
  • עגלת קניות מותאמת אישית – נבנתה מאפס בקוד פנימי ללא שימוש בספריות צד שלישי
  • אינטגרציה עם PayPal ותשלום בכרטיס אשראי
  • פריסה אוטומטית וחיבור דומיין מותאם אישית ב-Vercel
  • מערכת קופונים ומבצעים עם לוגיקה דינמית על בסיס כמות, ערך או סוג משתמש
  • לוח בקרה BI להצגת מדדים וסטטיסטיקות (מכירות, רווח, מלאי) באמצעות Recharts
  • ניהול משתמשים והרשאות מותאם אישית – נבנה מאפס בקוד פנימי ללא שימוש בספריות צד שלישי

4. סטאק טכנולוגי

  • Frontend: React 18, Next.js 14, Tailwind CSS.
  • Backend: Node.js 22, Firebase Auth & Firestore.
  • Deployment: Vercel CI/CD / Railway.
  • Storage: Firebase Storage / CDN.

5. תפקידי משתמשים

  • Client: Customer UI ו-Checkout Custom Made.
  • Admin: System Logs, Sentry Error Monitoring ו-Analytics.

  • פלוס 3,500₪ – התאמות UI ו-Support Services.

  • אינטגרציה ישירה עם PayPal ותשלום בכרטיס אשראי – קלות שימוש ואמינות
  • ניהול מוצרים מתקדם ב-PostgreSQL: מלאי בזמן אמת, וריאנטים, תמונות מרובות ויבוא/ייצואו
  • מנוע קופונים ודינמיק מבצעים – לוגיקה גמישה לפי כמות, ערך וסוג משתמש
  • פריסה מהירה וחיבור דומיין אוטומטי ב-Vercel ללא צורך בידע טכני
