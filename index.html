ReadySetStores - איפיון פרוייקט

פלטפורמה לבניית וניהול חנויות אונליין

1. סקירה טכנית

מערכת SPA ב-React 18 ו-Next.js 14 עם שילוב SSR/CSR לאופטימיזציית SEO וטעינה מהירה.
עיצוב Frontend ב-Tailwind CSS, ו-Backend ב-Firebase Auth, Firestore ו-Node.js 18.

2. מטרות מרכזיות

  • זמני טעינה <2 שניות (Lighthouse ≥90).
  • תמיכה ב-PWA עם Service Workers ו-Caching חכם.
  • דפי מוצר דינמיים בעזרת SSG/ISR של Next.js.
  • Scale אופקי מלא ב-Vercel/Railway.

3. תכונות עיקריות

  • טמפלייטים UI מבוססי JSON עם עדכון בזמן אמת.
  • ניהול מוצרים: images, variants[], price, inventory ב-Firestore.
  • עידכוני הזמנות בזמן אמת באמצעות Listeners.
  • אינטגרציה עם תשלום ישראלי (ביט, PayMe, ישראכרט).
  • חיבור דומיין אוטומטי דרך DNS/CNAME.
  • ניוזלטרים באמצעות Firebase Functions.
  • קופונים בדינמיק לוגיק ב-Cloud Functions.
  • דשבורד BI ל-Metrics עם Recharts.

4. סטאק טכנולוגי

  • Frontend: React 18, Next.js 14, Tailwind CSS.
  • Backend: Node.js 18, Firebase Auth & Firestore.
  • Deployment: Vercel CI/CD / Railway.
  • Storage: Firebase Storage / CDN.

5. תפקידי משתמשים

  • Client: Customer UI ו-Checkout Custom Made.
  • Admin: System Logs, Sentry Error Monitoring ו-Analytics.

6. תמחור

  • בסיס 2,500₪ – תבנית סטנדרטית.
  • פלוס 3,500₪ – התאמות UI ו-Support 1.
  • פרימיום 5,000₪ – עיצוב ממותג ו-Support 2.

דמי תחזוקה:

  • מינימום 30₪/חודש.
  • 50% מהעלות ללקוחי בסיס/פלוס.
  • 30% ללקוחות פרימיום.

7. יתרונות תחרותיים

  • RTL מובנה ורכיבים Configurable.
  • Bundle קטן <200KB אחרי Tree-shaking.
  • תיעוד API ו-SDK מובנה.

8. תוכניות עתידיות

  • Marketplace לתבניות UI.
  • REST & GraphQL API למפתחים.
  • כלי Advanced Marketing – Popups, Timers, A/B Testing.

9. סטטוס נוכחי

v1.0-alpha Frontend יציב, v0.5-beta Admin Dashboard. פגישה קרובה ל-Review שינויים.
