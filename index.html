איפיון טכני – ReadySetStores

בעלים: אריאל אלקסלאסי, אוריי ריספאלד

קהל יעד: עסקים קטנים ובינוניים בישראל

1. סקירה טכנית

מערכת SPA מבוססת React 18 ו-Next.js 14 עם SSR ו-CSR לשילוב SEO וטעינה מהירה. ה-frontend מעוצב באמצעות Tailwind CSS 3.0. ה-backend משתמש ב-Firebase Authentication, Firestore (v10 SDK) לאחסון מבני נתונים, ו-Node.js 18 לפיתוח פונקציות Serverless.

2. מטרות מרכזיות

  • זמני טעינה <2 שניות (Lighthouse Performance ≥ 90).
  • תמיכה ב-PWA עם Service Workers ו-Caching אוטומטי.
  • גרסת דפי מוצר דינמית באמצעות SSG/ISR של Next.js.
  • יכולת Scale אופקי בענן (Vercel/Railway).

3. תכונות עיקריות

  • טמפלייטים דינמיים ל-UI הניתנים להגדרה בקבצי JSON.
  • ניהול מוצרים: Collection “products” ב-Firestore עם שדות images, variants[], price, inventory.
  • עדכון סטטוס הזמנות בזמן אמת בעזרת Firestore listeners.
  • אינטגרציה עם API של ספקי תשלום ידועיים וביטוחים (כרגע פייפאל ואשראי).
  • DNS ו-CNAME לחיבור דומיין מותאם אוטומטי.
  • API לאיסוף כתובות מייל ושליחת ניוזלטרים באמצעות Firebase Functions.
  • מערכת קופונים מבוססת Business Logic ב-Cloud Functions.
  • דשבורד BI להצגת Metrics (pageViews, conversionRate, revenue) באמצעות Recharts.

4. סטאק טכנולוגי

  • Frontend: React 18, Next.js 14, Tailwind CSS 3
  • Backend: Node.js 18, Firebase Auth & Firestore v10
  • Deployment: Vercel (CI/CD) / Railway
  • Storage: Firebase Storage / CDN (Cloudflare)

5. תפקידי משתמשים

  • Owner: CRUD מלא על Firestore collections, הגדרות store דרך Admin Panel.
  • Client: Customer-Facing UI, Checkout flow באמצעות Stripe API.
  • Admin: Dashboard למעקב System Logs, Error Monitoring (Sentry) ו-Analytics.

6. תמחור

תשלום חד-פעמי

  • בסיס (2,500 ₪) – תבנית סטנדרטית.
  • פלוס (3,500 ₪) – תבנית עם התאמות UI ו-Support Level 1.
  • פרימיום (5,000 ₪) – עיצוב מותאם, Support Level 2, שדרוג רוחב פס.

דמי תחזוקה חודשיים

  • מינימום 30 ₪/חודש.
  • לקוחות בסיס/פלוס: 50% מהעלות החודשית.
  • לקוחות פרימיום: 30% מהעלות החודשית.

7. יתרונות תחרותיים

  • RTL מובנה בממשק וברכיבים.
  • Bundle גודל <200 KB אחרי Tree-shaking.
  • Components מודולריים ו-Configurable.
  • תיעוד API ו-SDK למפתחים.

8. תוכניות עתידיות

  • הוספת Marketplace לתבניות עיצוב.
  • ממשק מתכנתים (REST & GraphQL API).
  • כלי Advanced Marketing: Popups, Timers, A/B Testing.
  • פיתוח שיתוף קהילתי של תבניות ומשאבים.

9. סטטוס נוכחי

Client v1.0-alpha (Frontend יציב), Admin Dashboard v0.5-beta. פגישה שלישית עם הלקוח ל-Review שינויים מתוכננת בקרוב.
